Research Experience

Bachelor Thesis | Supervisor: Dr. MohammadReza Jahed-Motlagh Oct 2025

A System for Psychological Stress Assessment Based on Text

  • Innovated in data quality improvement by re-labeling a large dataset of Persian tweets using the Gemini 2.5 Flash model with a sophisticated, engineered prompt.
  • Created a high-quality, refined dataset of 4.2 million samples by filtering for instances where the original label and the new LLM-generated label were in agreement.
  • Fine tuned and trained a diverse set of models on this refined data, including classic (Logistic Regression, SVM), word representation (FastText), and Transformer-based (a Persian BERT) models.
  • Implemented an Ensemble Learning model using a weighted-voting strategy to combine the predictions from all four models for the final assessment.
  • Built a complete, end-to-end web application featuring a Flask backend (to serve the ensemble model via an API) and an HTML/CSS/JavaScript frontend.
  • The final application provides users with a stress analysis, including Chart.js visualizations for both "sectional" (local, 3-word chunks) and "cumulative" (progressive, overall) stress levels in the text.
  • Conducted a detailed results analysis, proving that data refinement dramatically improved model performance.
